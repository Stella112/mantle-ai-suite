# Mantle AI Suite 🛡️📊🔍

**An autonomous, modular AI ecosystem built exclusively for the Mantle Network.**

The **Mantle AI Suite** is a collection of agentic security and community intelligence tools designed to solve the biggest existential risks in Web3: transaction safety, community authenticity, and technical accessibility.

---

## 🚀 The Three Pillars

### 1. 🛡️ Mantle Aegis (The Enforcer)
**Real-time Transaction Shield.** Aegis is an autonomous AI agent that intercepts malicious transaction prompts before you sign. 
- **TEE Auditing**: Simulates transactions in a Trusted Execution Environment to identify hidden malicious functions like `clearETH`.
- **Closed Learning Loop**: Continuously ingests failed transactions and mempool data to detect zero-day exploits.
- **Gasless Oracle**: Anchors risk assessments on-chain using Mantle's TSS-node infrastructure.

### 2. 📊 MantlePulse AI (The community Analyst)
**Sybil-Resistant On-chain CRM.** Pulse analyzes wallet behavior and cross-references it with social sentiment to reward true community members.
- **True Fan Score**: Generates individual scores based on transaction volume, contract depth, and off-chain social proof.
- **Auto-Yield Routing**: Developers can plug Pulse into smart contracts to automate $MNT rewards strictly for verified humans.

### 3. 🔍 MantleGuard AI (The Security Analyst)
**Plain-English Smart Contract Auditor.** A conversational interface for retail users to audit protocols without reading Solidity.
- **Instant Audits**: Ask "Is this DEX safe?" and get an instant breakdown of contract logic and dev history.
- **Hype vs. Utility**: Sophisticated NLP models score tokens based on actual on-chain utility vs. social hype.

## 🏗️ Technical Architecture

The Mantle AI Suite leverages the unique modular architecture of the **Mantle Network**:

- **Mantle DA (EigenDA)**: Used for high-capacity, low-cost storage of historical threat vectors and CRM sentiment data.
- **L2 Scalability**: Real-time transaction interception and on-the-fly audits require the sub-second finality and microscopic fees only Mantle provides.
- **TSS Infrastructure**: Decentralized risk oracles are anchored via Mantle’s Threshold Signature Scheme nodes to ensure trust-minimized reporting.
